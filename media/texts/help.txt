Basic functions:
------------------
ESC         : Exit Computer / Exit Game
Q S D Z A E : Player move
Arrows      : Player move
RIGHT-SHIFT : Player run
Mouse Clic  : Open Door, Activate Robot, etc
F1          : This help screen
.
TAB         : Inventory (not implemented yet)
Right Clic  : Move head, when using a computer (not implemented yet)
.
Other functions:
-----------------
F12         : Show FrameRate
PrtScr      : Take a Screenshot
T           : Change rendering (bilinear, trilinear, anisotropic)
.
.
