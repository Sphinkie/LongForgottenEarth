Long Forgotten Earth

Developper Corner:

We provide a Doxygen documentation explaining the C++ classes of the source code.
The star positions are all real, and come from the Celestia database.
The detailled characteristics of solar systems (star type, planets, atmosphere, temperature, etc) are realistic too, and generated by StarGen II.
Other usefull documentation and scripts can be found on this GitHub sub-repository.
