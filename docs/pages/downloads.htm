Installation Notes

Create a directory: \sphinx (or the name you want)
Copy the LFEsetup.exe files and run it.
You should then have these three folders:
- \sphinx\Release
- \sphinx\media
- \sphinx\savedgames
Go to \Release, and run config.exe for configuration, then run sphinx.exe to start playing te game.
