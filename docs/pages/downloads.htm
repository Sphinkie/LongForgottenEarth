Installation Note

Create a directory: \sphinx (or the name you want)
Copy LFEsetup.exe inside and run it.
You should then have three folders:
- \sphinx\Release
- \sphinx\media
- \sphinx\savedgames
Go to \Release: first run config.exe for configuration, then run sphinx.exe to start playing the game.

Enjoy !
